Suporte dos Navegadores

Tag Descrição
<a> A tag <a> define um hyperlink, que é usada para linkar uma página em outra.
<abbr> A tag <abbr> define uma abreviação ou um acrônimo, como "HTML", "Sr.", "Sra.", "Prof.".
<address> A tag <address> define as informações de contato do autor/proprietário de um documento ou artigo.
<article> A tag <article> especifica um conteúdo independente.
<aside> A tag <aside> define algum conteúdo a parte do conteúdo em que ela é colocada.
<audio> A tag <audio> define som, como músicas dentre outras.
<b> A tag <b> especifica um texto em negrito sem nenhuma importância extra.
<base> A tag <base> especifica a URL/alvo base para todas relativas URL's em um documento.
<bdi> A tag <bdi> isola uma parte do texto que deve ser formatada em uma direção diferente de outro texto externo.
<bdo> A tag <bdo> é usada para sobrescrever a direção atual do texto.
<blockquote> A tag <blockquote> especifica um trecho que foi citado de uma outra fonte.
<body> A tag <body> define o corpo do documento.
<br> A tag <br> insere uma única quebra de linha.
<button> A tag <button> define um botão clicável.
<caption> A tag <caption> define o título de uma tabela.
<code> A tag <code> é uma tag de expressão que define um pedaço de código de computador.
<col> A tag <col> especifica propriedades para cada coluna com a tag <colgroup>.
<colgroup> A tag <colgroup> especifica um grupo de uma ou mais colunas em uma tabela para formatação.
<del> A tag <del> define um texto que foi deletado de um documento.
<details> A tag <details> especifica detalhes adicionais que o usuário pode visualizar ou esconder.
<div> A tag <div> define uma divisão ou uma seção em um documento HTML.
<em> A tag <em> é uma tag frasal. Renderizará um texto enfatizado.
<fieldset> A tag <fieldset> é usado é usada para agrupar elementos relacionados em um formulário.
<figcaption> A tag <figcaption> define o título para uma tag <figure>.
<figure> A tag <figure> especifica um conteúdo independente, como ilustrações, diagramas, fotos, etc.
<footer> A tag <footer> define um rodapé para um documento ou uma seção.
<form> A tag <form> é usada para criar um formulário HTML para entrada de dados de usuário.
<h1 a h6> As tags <h1> até a tag <h6> são usadas para definir títulos.
<head> A tag <head> é um container para todos elementos do cabeçalho.
<header> A tag <header> representa um container para um conteúdo introdutório ou um conjunto de links.
<hr> A tag <hr> define uma parada de um tema em uma página HTML (por exemplo uma mudança para um assunto fora de contexto).
<html> A tag <html> informa ao navegador que isso é um documento HTML.
<i> A tag <i> define uma parte de um texto de uma forma alternativa.
<img> A tag <img> define uma imagem em uma página HTML.
<input> A tag <input> especifica um campo de entrada onde usuários podem inserir dados.
<ins> A tag <ins> define um texto que foi inserido em um documento.
<kbd> A tag <kbd> é uma tag frasal. Ela define entrada do teclado.
<label> A tag <label> define um rótulo para vários elementos.
<legend> A tag <legend> define um título para um elemento <fieldset>.
<li> A tag <li> define um item de lista.
<link> A tag <link> define um link entre um documento e um recurso externo.
<main> A tag <main> especifica o conteúdo principal de um documento.
<mark> A tag <mark> define um texto marcado. Use a tag <mark> se você precisa destacar partes de seu texto.
<meta> A tag <meta> fornece metadata sobre um documento HTML.
<nav> A tag <nav> define um grupo de link de navegação.
<ol> A tag <ol> define uma lista ordenada.
<option> A tag <option> define uma opção em uma lista de seleção.
<output> A tag <output> representa o resultado de um cálculo (realizado por um script por exemplo).
<p> A tag <p> define um parágrafo.
<picture> O uso mais comum da tag <picture> será para direção de artística em designers responsivos.
<pre> A tag <pre> define um texto pré-formatado.
<progress> A tag <progress> representa o progresso de uma tarefa.
<q> A tag <q> define uma curta citação.
<script> A tag <script> é usada para definir script do lado do cliente (JavaScript).
<section> A tag <section> define seções em um documento.
<select> A tag <select> é usada para criar uma lista drop-down.
<source> A tag <source> é usada para especificar vários recursos de mídia para elementos de mídia.
<span> A tag <span> é usada para agrupar elementos embutidos em um documento.
<strong> A tag <strong> é uma tag frasal. Ele define texto importante.
<style> A tag <style> é usada para definir informações de estilo para um documento HTML.
<sub> A tag <sub> define o texto subscrito.
<summary> A tag <summary> define um cabeçalho visível para o elemento <details>.
<sup> A tag <sup> define o texto sobrescrito.
<table> A tag <table> define uma tabela HTML.
<tbody> A tag <tbody> é usada para agrupar o conteúdo do corpo em uma tabela HTML.
<td> A tag <td> define uma célula padrão em uma tabela HTML.
<textarea> A tag <textarea> define um controle de entrada de texto com várias linhas.
<tfoot> A tag <tfoot> é usada para agrupar o conteúdo do rodapé em uma tabela HTML.
<th> A tag <th> define uma célula de cabeçalho em uma tabela HTML.
<thead> A tag <thead> é usada para agrupar o conteúdo do cabeçalho em uma tabela HTML.
<title> A tag <title> é necessária em todos os documentos HTML e define o título do documento.
<tr> A tag <tr> define uma linha em uma tabela HTML.
<u> A tag <u> representa algum texto que deve ser estilisticamente diferente do texto normal.
<ul> A tag <ul> define uma lista não ordenada (com marcadores).
<video> A tag <video> especifica o vídeo, como um clipe de filme ou outras transmissões de vídeo.
