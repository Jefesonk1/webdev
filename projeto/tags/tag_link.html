Definição e Uso

A tag <link> define um link entre um documento e um recurso externo.
A tag <link> é usada para linkar folhas estilo externa.

Exemplo

Link para um folha de estilo externa:

            
                <head>
                    <link rel="stylesheet" type="text/css" href="theme.css">
                </head>

Atributos

no-referrer-when-downgrade
origin
origin-when-cross-origin
