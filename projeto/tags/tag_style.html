Definição e Uso

A tag <style> é usada para definir informações de estilo para um documento HTML.
Dentro do elemento <style>, você especifica como os elementos HTML devem renderizar em um navegador.
Cada documento HTML pode conter várias tags <style>.

Exemplo

Uso do elemento <style> em um documento HTML:

            
                <html>
                    <head>
                        <style>
                            h1 {color:red;}
                            p {color:blue;}
                        </style>
                    </head>
                    <body>

                        <h1>A heading</h1>
                        <p>A paragraph.</p>

                    </body>
                </html>

Atributos

Atributos Valor Descrição
media media_query Especifica qual mídia/dispositivo o recurso é otimizado
type text/css Especifica o tipo de mídia da tag <style>
