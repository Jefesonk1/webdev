Definição e Uso

A tag <video> especifica o vídeo, como um clipe de filme ou outras transmissões de vídeo.
Atualmente, existem 3 formatos de vídeo suportados para o elemento <video>: MP4, WebM e Ogg:

Navegador MP4 WebM Ogg

Chrome SIM SIM SIM

  • MP4 = arquivos MPEG 4 com codec de vídeo H264 e codec de áudio AAC
  • WebM = arquivos WebM com codec de vídeo VP8 e codec de áudio Vorbis
  • Ogg = arquivos Ogg com codec de vídeo Theora e codec de áudio Vorbis

MIME Types para formatos de áudio

Formato MIME-type
MP4 video/mp4
WebM video/webm
Ogg video/ogg

Exemplo

Reproduzir um vídeo:

            
                <video width="320" height="240" controls>
                    <source src="movie.mp4" type="video/mp4">
                    <source src="movie.ogg" type="video/ogg">
                    Your browser does not support the video tag.
                </video>

Atributos

Atributos Valor Descrição
autoplay autoplay Especifica que o vídeo começará a ser reproduzido assim que estiver pronto
controls controls Especifica que os controles de vídeo devem ser exibidos (como um botão de reprodução / pausa etc.)
height pixels Define a altura do player de vídeo
loop loop Especifica que o vídeo será reiniciado novamente, sempre que terminar
muted muted Especifica que a saída de áudio do vídeo deve ser silenciada
poster URL Especifica uma imagem a ser exibida enquanto o vídeo está sendo baixado ou até o usuário clicar no botão de reprodução
preload auto
metadata
none		 Especifica se e como o autor pensa que o vídeo deve ser carregado quando a página for carregada
src URL Especifica o URL do arquivo de vídeo
width pixels Define a largura do player de vídeo
