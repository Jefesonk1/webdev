Verificar o Suporte dos Navegadores

Definição e Uso

A tag <q> define uma curta citação.
Os navegadores geralmente inserem marcas de citação ao redor de uma citação.

Exemplo

Marca uma breve citação:

            
                <p>WWF's goal is to:
                <q>Build a future where people live in harmony with nature.</q>
                We hope they succeed.</p>

Atributos

Atributos Valor Descrição
cite URL Especifica a URL fonte da citação
