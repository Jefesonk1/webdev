Definição e Uso

A tag <figcaption> define o título para uma tag <figure>.
A tag <figcaption> pode ser colocada como o primeiro ou último filho de uma tag <figure>.

Exemplo

Use a tag <figure> para colocar um foto em um documento, e a tag <figcaption> para definir um título para a ela:

            
                <figure>
                    <img src="pic_trulli.jpg" alt="Trulli" style="width:100%">
                    <figcaption>Fig.1 - Trulli, Puglia, Italy.</figcaption>
                </figure>

Atributos

A tag <figcaption> suporta os Atributos Globais e os Atributos de Eventos.
