Definição e Uso

A tag <em> é uma tag frasal. Renderizará um texto enfatizado.
Dica: Essa tag não está obsoleta, mas é possível alcançar um melhor efeito utilizando CSS.
Todas as tags frasais:

Tag Description
<em> Renderiza um texto enfatizado
<strong> Define um texto importante
<code> Define um pedaço de código de computador
<samp> Define um exemplo de saída de um programa de computador
<kbd> Define uma entrada de teclado
<var> Define uma variável

Exemplo

Formata o texto em um documento:

            
                <em>Emphasized text</em>

Atributos

A tag <em> suporta os Atributos Globais e os Atributos de Eventos.
