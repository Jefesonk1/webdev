Definição e Uso

A tag <del> define um texto que foi deletado de um documento.

Exemplo

Define a cor de fnda de três colunas com a tags <colgroup> e <col>:

            
                <p>My favorite color is <del>blue</del> <ins>red</ins>!</p>

Atributos

Atributo Valor Descrição
cite URL Especifica a URL para um a document que explica o motivo do texto ser deletado
datetime YYYY-MM-DDThh:mm:ssTZD Especifica a data e a hora de quando o texto foi deletado
