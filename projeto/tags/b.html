Definição e Uso

The <b> tag specifies bold text without any extra importance.

Exemplo

Make text bold:

            
                <p>This is normal text - <b>and this is bold text</b>.</p>

Atributos

The <b> tag supports the Global Attributes and Event Attributes in HTML.
