Suporte dos Navegadores

Definição e Uso

A tag <ins> define um texto que foi inserido em um documento.
Dica: Observe também a tag <del> para marcar um texto excluído.
Os navegadores geralmente exibirão uma linha horizontal no meio do texto excluído e uma linha vertical abaixo do texto inserido.

Exemplo

Um texto com uma parte excluída e uma nova parte inserida:

            
                <p>My favorite color is <del>blue</del> <ins>red</ins>!</p>

Atributos

A tag <ins> suporta os Atributos Globais e os Atributos de Eventos.
