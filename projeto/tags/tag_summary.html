Suporte dos Navegadores

Definição e Uso

A tag <summary> define um cabeçalho visível para o elemento <details> . É possível clicar no cabeçalho para visualizar / ocultar os detalhes.

Exemplo

Usando o elemento <summary>:

            
                <details>
                    <summary>Copyright 1999-2014.</summary>
                    <p> - by Refsnes Data. All Rights Reserved.</p>
                    <p>All content and graphics on this web site are the property of the company Refsnes Data.</p>
                </details>

Atributos

A tag <summary> suporta os Atributos Globais e os Atributos de Eventos.
