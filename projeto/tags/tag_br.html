Definição e Uso

A tag <br> insere uma única quebra de linha.
A tag <br> é uma tag vazia, o que significa que ela não possui uma tag final.

Exemplo

A tag <br> inserre uma única quebra de linha;

            
                <p>To force<br> line breaks<br> in a text,<br> use the br<br> element.</p>

Atributos

A tag <br> suporta os Atributos Globais e os Atributos de Eventos.
