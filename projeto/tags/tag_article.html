Definição e Uso

A tag <article> especifica um conteúdo independente.
Um artigo deve fazer sentido por si só, e ser possível distribui-lo independentemente do restante do site.
Potencias utilizatções para tag <article>:

  • Post em Fórum
  • Post em Blog
  • Notícias
  • Comentário

Exemplo

Dois artigos com conteúdos independentes:

            
                <article>
                    <h2>Google Chrome</h2>
                    <p>Google Chrome is a web browser developed by Google, 
                    released in 2008. Chrome is the world's most popular web 
                    browser today!</p>
                </article>
                
                <article>
                    <h2>Microsoft Edge</h2>
                    <p>Microsoft Edge is a web browser developed by Microsoft, released in 2015.
                    Microsoft Edge replaced Internet Explorer.</p>
                </article>

Atributos

A tag <article> suporta os Atributos Globais e os Atributos de Eventos.
