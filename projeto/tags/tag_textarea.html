Definição e Uso

A tag <textarea> define um controle de entrada de texto com várias linhas.
Uma área de texto pode conter um número ilimitado de caracteres e o texto é renderizado em uma fonte de largura fixa (geralmente Courier).
O tamanho de uma área de texto pode ser especificado pelos atributos colunas e linhas, ou melhor ainda; pelas propriedades de altura e largura do CSS.
Dica: sempre adicione a tag <label> para melhor acessibilidade!

Exemplo

Uma área de texto HTML:

            
                <label for="w3mission">W3Schools mission:</label>

                <textarea id="w3mission" rows="4" cols="50">
                    At w3schools.com you will learn how to make a website. We offer free tutorials in all web development technologies.
                </textarea>

Atributos

Atributos Valor Descrição
autofocus autofocus Especifica que uma área de texto deve obter foco automaticamente quando a página é carregada
cols number Especifica a largura visível de uma área de texto
dirname textareaname.dir Especifica que a direção do texto da área de texto será submetido
disabled disabled Especifica que uma área de texto deve ser desativada
form form_id Especifica a qual formulário a área de texto pertence.
maxlength number Especifica o número máximo de caracteres permitido na área de texto
name text Especifica um nome para uma área de texto
placeholder text Especifica uma dica curta que descreve o valor esperado de uma área de texto
readonly readonly Especifica que uma área de texto deve ser somente leitura
required required Especifica que uma área de texto é necessária / deve ser preenchida
rows number Especifica o número visível de linhas em uma área de texto
wrap hard
soft		 Especifica como o texto em uma área de texto deve ser quebrado quando enviado em um formulário
