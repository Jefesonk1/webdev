Definição e Uso

A tag <input> especifica um campo de entrada onde usuários podem inserir dados.
Elementos <input> são usados dentro de uma tag <form> para declarar controles de entrada que permitem que usuários insiram dados.
Um campo de entrada pode variar, dependendo do tipo do atributo.

Exemplo

Um formulário HTML com três campos de entrada; dois campos de texto e um botão de submissão:

            
                <form action="/action_page.php">
                    <label for="fname">First name:</label>
                    <input type="text" id="fname" name="fname"><br><br>
                    <label for="lname">Last name:</label>
                    <input type="text" id="lname" name="lname"><br><br>
                    <input type="submit" value="Submit">
                </form>
