Definição e Uso

BDI serve para isolação bidirecional.
A tag <bdi> isola uma parte do texto que deve ser formatada em uma direção diferente de outro texto externo.
Esse elemento é útil quando existe um conteúdo gerado por um usuário e possui uma direção desconhecida.

Exemplo

Isola os nomes de usuários em uma direção diferente:

            
                <ul>
                    <li>User <bdi>hrefs</bdi>: 60 points</li>
                    <li>User <bdi>jdoe</bdi>: 80 points</li>
                    <li>User <bdi>إيان</bdi>: 90 points</li>
                </ul>

Atributos

A tag <bdi> suporta os Atributos Globais e os Atributos de Eventos.
