Definição e Uso

A tag <button> define um botão clicável.
Dentro de uma tag <button>, você pode inserir conteúdos, como textos ou imagens. Essa é a diferença entre essa tag e os botões criados com a tag <input>.
Dica: Sempre especifique o tipo de atributo para um <button>. Diferentes navegadores utilizam diferentes padrões de tipos para o elemento <button>.
DIca: Botões são frequentimente estilizados com CSS.

Exemplo

Um botão clicável é exibido a seguir:

            
                <button type="button">Click Me!</button>

Atributos

Atributos Valor Descrição
autofocus autofocus Especifica que o botão deve automaticamente receber foco quando a página carrega
disabled disabled Especifica que o botão deve ser desativado
form form_id Especifica a qual formulário o botão pertence
formaction URL Especifica onde enviar os dados do formulário quando ele é submetido. Apenas para type="submit"
formenctype application/x-www-form-urlencoded
multipart/form-data
text/plain		 Especifica como os dados do formulário deve ser codificado antes de ser enviado ao servidor. Apenas para type="submit"
formmethod get
post		 Especifica como enviar os dados de um formulário (qual método HTTP a ser usado). Apenas para type="submit"
formnovalidate formnovalidate Especifica que os dados de um formulário não deve ser validado na submissão. Apenas para type="submit"
formtarget _blank
_self
_parent
_top
framename		 Especifica onde exibir a resposta depois de submeter o formulário. Apenas para type="submit"
name name Especifica um nome para um botão
type button
reset
submit		 Especifica o tipo de um botão
value text Especifica um valor inicial para um botão
