Suporte dos Navegadores

Definição e Uso

A tag <audio> define som, como músicas dentre outras.
Atualmente, 3 formatos de arquivos são suportados: MP3, WAV, e OGG:

Navegadores MP3 WAV OGG
Internet Explorer YES NO NO
Chrome YES YES YES
Firefox YES YES YES
Safari YES YES NO
Opera YES YES YES

MIME Types para formatos de áudio

Formato MIME-type
MP3 audio/mpeg
OGG audio/ogg
WAV audio/wav

Exemplo

Tocar um som:

            
                <audio controls>
                    <source src="horse.ogg" type="audio/ogg">
                    <source src="horse.mp3" type="audio/mpeg">
                    Your browser does not support the audio tag.
                </audio>

Atributos

Atributos Valor Descrição
autoplay autoplay Especifica que o aúdio será reproduzido assim que estiver pronto
controls controls Especifica quais controles de áudios serão exibidos (como botões de play/pause etc)
loop loop Especifica que o áudio começará novamente, toda vez que terminar
muted muted Especifica que a saída do aúdio deve ser muda
preload auto
metadata
none		 Especifica se e como o autor do áudio imagina que o áudio seja carregado quando a página carrega
src URL Especifica a URL do arquivo de áudio
