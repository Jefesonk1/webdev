Definição e Uso

A tag <main> especifica o conteúdo principal de um documento.
O conteúdo dentro da tag <main> deve ser único para o documento. Ele não deve conter qualquer conteúdo que é repetido entre documentos, como barras laterais, links de navegação, informações de direitos autorais, logos, e formulários de pesquisa.
Nota: Não deve existir mais de uma tag <main> em um documento. A tag <main> não deve ser usada dentro de das tags <article>, <aside>, <footer>, <header>, ou <nav>.

Exemplo

Exemplo:

            
                <main>
                    <h2>Web Browsers</h2>
                    <p>Chrome, Firefox, and Edge are the most used browsers today.</p>
                
                    <article>
                        <h3>Google Chrome</h3>
                        <p>Chrome is a free, open-source web browser developed by Google, released in 2008.</p>
                    </article>
                
                    <article>
                        <h3>Microsoft Edge</h3>
                        <p>Edge is a free web browser from Microsoft, released in 1995.</p>
                    </article>
                
                    <article>
                        <h3>Mozilla Firefox</h3>
                        <p>Firefox is a free, open-source web browser from Mozilla, released in 2004.</p>
                    </article>
                </main>

Atributos

A tag <main> suporta os Atributos Globais e os Atributos de Eventos.
