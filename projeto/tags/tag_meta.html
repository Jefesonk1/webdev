Definição e Uso

Metadata são dados (informação) sobre dados.
A tag <meta> fornece metadata sobre um documento HTML. Metadata não será exibida na página, mas será analisada pela máquina.
Elementos meta são tipicamente usados para especificar descrição de página, palavras-chave, autor do documento, última modificação e outras.
Metadata pode ser usado por navegadores (como exibir um conteúdo ou recarregar página), motores de busca (palavras-chave) ou outros serviços.
HTML5 introduziu um método para deixar web designer controlar a viewport ( a área visível de uma página) através da tag <meta>.

Exemplo

Exemplo 1 - Define palavras-chave para motores de busca:

            
                <meta name="keywords" content="HTML, CSS, XML, XHTML, JavaScript">

Exemplo 2 - Define a descrição da sua página web:

            
                <meta name="description" content="Free Web tutorials on HTML and CSS">

Exemplo 3 - Define o autor da página:

            
                <meta name="author" content="John Doe">

Exemplo 4 - Recarrega o documendo a cada 30 segundos:

            
                <meta http-equiv="refresh" content="30">

Exemplo 5 - Configura a viewport para fazer seu website ser bem exibido em todos os dispositivos:

            
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
