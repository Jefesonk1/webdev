Definição e Uso

A tag <kbd> é uma tag frasal. Ela define entrada do teclado.
Dica: Essa tag não está obsoleta, mas é possível obter um melhor efeito utilizando CSS.
Todas as tags frasais:

Tag Description
<em> Renderiza um texto enfatizado
<strong> Define um texto importante
<code> Define um pedaço de código de computador
<samp> Define um exemplo de saída de um programa de computador
<kbd> Define uma entrada de teclado
<var> Define uma variável

Exemplo

Formata um texto em um documento:

            
                <kbd>Keyboard input</kbd>

Atributos

A tag <kbd> suporta os Atributos Globais e os Atributos de Eventos.
