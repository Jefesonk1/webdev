Definição e Uso

A tag <figure> especifica um conteúdo independente, como ilustrações, diagramas, fotos, etc.
Enquanto o conteúdo de uma tag <figure> é relacionada com o fluxo principal, sua posição é independente do mesmo, e se removida, não deverá afetar o fluxo do documento.

Exemplo

Use a tag <figure> para colocar um foto em um documento, e a tag <figcaption> para definir um título para a ela:

            
                <figure>
                    <img src="pic_trulli.jpg" alt="Trulli" style="width:100%">
                    <figcaption>Fig.1 - Trulli, Puglia, Italy.</figcaption>
                </figure>

Atributos

A tag <figure> suporta os Atributos Globais e os Atributos de Eventos.
