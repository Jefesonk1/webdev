Definição e Uso

A tag <img> define uma imagem em uma página HTML.
A tag <img> tem dois atributos obrigatórios: src e alt.
Nota: Imagens não são tecnicamente inseridas em uma página HTML, imagens são linkadas a elas. A tag <img> cria um espaço reservado para a imagem referenciada.
Dica: Para linkar uma imagem a outro documento, simplesmente insira a tag <img> dentro da tag <a>.

Exemplo

Como inserir uma imagem:

            
                <img src="smiley.gif" alt="Smiley face" height="42" width="42">

Atributos

Atributos Valor Descrição
alt text Especifica um texto alternativo para uma imagem
crossorigin anonymous
use-credentials		 Permite imagens de sites de terceiros que permitem origem cruzada
height pixels Especifica a altura de uma imagem
ismap ismap Especifica uma imagem como uma server-side image-map
longdesc URL Especifica a URL para uma descrição detalhada de uma imagem
referrerpolicy no-referrer
no-referrer-when-downgrade
origin
origin-when-cross-origin
unsafe-url		 Especifica qual referenciador usar quando buscarem a imagem
sizes sizes Especifica o tamanho da imagem para diferentes layouts de página
src URL Especifica a URL da imagem
srcset URL Especifica a URL da imagem para usar em diferentes situações
usemap #mapname Especifica uma imagem como uma client-side image-map
width pixels Especifica a largura de uma imagem
