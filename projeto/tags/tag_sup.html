Definição e Uso

A tag <sup> define o texto sobrescrito. O texto sobrescrito aparece com meio caractere acima da linha normal e às vezes é renderizado em uma fonte menor.
O texto sobrescrito pode ser usado para notas de rodapé, como WWW[1].
Dica: Use a tag <sub> para definir o texto subscrito.

Exemplo

Texto sobrescrito:

            
                <p>This text contains <sup>superscript</sup> text.</p>

Atributos

A tag <sup> suporta os Atributos Globais e os Atributos de Eventos.
