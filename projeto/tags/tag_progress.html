Definição e Uso

A tag <progress> representa o progresso de uma tarefa.
Dica: Sempre adicione a tag <label> para melhor acessibilidade.

Exemplo

Exibe uma barra de progresso:

            
                <label for="file">Downloading progress:</label>
                <progress id="file" value="32" max="100"> 32% </progress>

Atributos

Atributos Valor Descrição
max number Especifica quanto tempo a tarefa requer no total. O valor padrão é 1
value number Especifica o quanto a tarefa já foi completada
