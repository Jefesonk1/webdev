Definição e Uso

A tag <address> define as informações de contato do autor/proprietário de um documento ou artigo.
Se a tag <address> estiver dentro da tag <body>, esta representará informações de contato para o documento.
Se a tag <address> estiver dentro da tag <article>, esta representará informações de contato para aquele artigo.
O texto na tag <address> geralmente renderizará em itálico. A maioria dos navegadores adicionará uma quebra de linha antes e depois da tag.

Exemplo

Informações de contato para Example.com:

            
                <address>
                    Written by <a href="mailto:webmaster@example.com">Jon Doe</a>.<br>
                    Visit us at:<br>
                    Example.com<br>
                    Box 564, Disneyland<br>
                    USA
                </address>

Atributos

A tag <address> suporta os Atributos Globais e os Atributos de Eventos.
