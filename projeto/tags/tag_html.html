Suporte dos Navegadores

Definição e Uso

A tag <html> informa ao navegador que isso é um documento HTML.
A tag <html> representa a raiz de um documento HTML.
A tag <html> é o container para todas as outras tags HTML (exceto pela tag <!DOCTYPE>).

Exemplo

Um simples documento HTML5:

            
                <!DOCTYPE HTML>
                <html>
                    <head>
                        <title>Title of the document</title>
                    </head>

                    <body>
                     The content of the document......
                    </body>

                </html>

Atributos

Atributos Valor Descrição
xmlns http://www.w3.org/1999/xhtml Especifica o atributo namespace XML (Se você precisa que a sintaxe do seu conteúdo seja XHTML).
