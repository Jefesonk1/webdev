Suporte dos Navegadores

Definição e Uso

A tag <nav> define um grupo de link de navegação.
Observe que nem todos os links de um documento devem estar dentro de uma tag <nav>. O elemento <nav> destina-se apenas ao bloco principal de links de navegação.
Navegadores, como leitores de tela para usuários deficientes, podem usar esse elemento para determinar se deve omitir a renderização inicial desse conteúdo.

Exemplo

Um grupo de links de navegação:

            
                <nav>
                    <a href="/html/">HTML</a> |
                    <a href="/css/">CSS</a> |
                    <a href="/js/">JavaScript</a> |
                    <a href="/jquery/">jQuery</a>
                 </nav>

Atributos

A tag <nav> suporta os Atributos Globais e os Atributos de Eventos.
