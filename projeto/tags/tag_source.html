Definição e Uso

A tag <source> é usada para especificar vários recursos de mídia para elementos de mídia, como <video>, <audio> e <picture>.
A tag <source> permite especificar arquivos de vídeo / áudio / imagem alternativos que o navegador pode escolher, com base em seu tipo de mídia, suporte a codec ou consulta de mídia.

Exemplo

Um reprodutor de áudio com dois arquivos de origem. O navegador deve escolher para qual arquivo (se houver) ele suporta:

            
                <audio controls>
                    <source src="horse.ogg" type="audio/ogg">
                    <source src="horse.mp3" type="audio/mpeg">
                    Your browser does not support the audio element.
                </audio>

Atributos

Atributos Valor Descrição
src URL Requerido quando <source> é usada em <audio> e <video>. Especifica a URL do arquivo de mídia
srcset URL Requerido quando <source> é usada em <picture>. Especifica a URL da imagem para usar em diferentes situações
media media_query Aceita qualquer requisição de mídia que normalmente seria definida em um CSS
sizes   Especifica tamanho de imagens para diferentes layouts de páginas
type MIME-type Especifica o MIME-type do recurso
