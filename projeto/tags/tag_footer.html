Suporte dos Navegadores

Definição e Uso

A tag <footer> define um rodapé para um documento ou uma seção.
A tag <footer> deve conter informações sobre seus elementos contidos.
A tag <footer> tipicamente contém;

  • Informações de autoria
  • Informações de direitos autorais
  • Informações de contato
  • Mapa do site
  • Link para o topo da página
  • Documentos relacionados

Você pode ter várias tags <footer> em um documento.

Exemplo

Uma seção com um rodapé em um documento:

            
                <footer>
                    <p>Posted by: Hege Refsnes</p>
                    <p>Contact information: <a href="mailto:someone@example.com">
                    someone@example.com</a>.</p>
                </footer>

Atributos

A tag <footer> suporta os Atributos Globais e os Atributos de Eventos.
