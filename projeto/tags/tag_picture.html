Definição e Uso

A tag <picture> da aos desenvolvedores web mais flexibilidade em especificar recursos de imagens.
O uso mais comum da tag <picture> será para direção de artística em designers responsivos. Ao invés de ter uma imagem que será escalada de acordo com o tamanho da viewport, múltiplas imagens podem ser designadas para um melhor resultado.
A tag <picture> carrega duas diferentes tags: uma ou mais tag <source> e uma tag <img>.
A tag <source> tem os seguintes atributos:

  • srcset (requerido) - define a URL da imagem a ser exibida
  • media - aceita qualquer consulta de mídia válida que normalmente seria definida em um CSS
  • sizes - define um descritor de largura única, uma consulta de mídia única com descritor de largura ou uma lista delimitada por vírgula de consultas de mídia com um descritor de largura
  • type - define o tipo MIME

O navegador usará os valores do atributo para carregar a imagem mais apropriada. O navegador usará o primeiro elemento <source> com uma dica correspondente e ignorará as seguintes tags <source>.
A tag <img> é requerida como a última tag filha da tag <picture>. O navegador usará a primeira tag <source> compatível como principal e ignorará as seguintes.
A tag <picture> trabalha de forma similar a tag <video> e a tag <audio>. Você define diferentes fontes, e a primeira fonte que se encaixar com a preferência é a ser usada.

Exemplo

Como usar a tag <picture>:

            
                <picture>
                    <source media="(min-width: 650px)" srcset="img_pink_flowers.jpg">
                    <source media="(min-width: 465px)" srcset="img_white_flower.jpg">
                    <img src="img_orange_flowers.jpg" alt="Flowers" style="width:auto;">
                </picture>

Atributos

A tag <picture> suporta os Atributos Globais e os Atributos de Eventos.
