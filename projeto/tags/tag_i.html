Definição e Uso

A tag <i> define uma parte de um texto de uma forma alternativa. O conteúdo de uma tag <i> é geralmente exibida em itálico.
A tag <i> pode ser usada para indicar um termo técnico, uma frase de outra língua, um pensamento, etc.
Use a tag <i> somente quando não há uma tag semântica mais apropriada, como:

  • <em> (texto enfatizado)
  • <strong> (texto importante)
  • <mark> (texto marcado/destacado)
  • <cite> (o título de um trabalho)
  • <dfn> (um termo de definição)

Exemplo

Exemplo:

            
                <p>He named his car <i>The lightning</i>, because it was very fast.</p>

Atributos

A tag <i> suporta os Atributos Globais e os Atributos de Eventos.
