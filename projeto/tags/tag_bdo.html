Definição e Uso

bdo serve para sobrescrita bidirecional.
A tag <bdo> é usada para sobrescrever a direção atual do texto.

Exemplo

Especifica a direção do texto:

            
                <bdo dir="rtl">
                    This text will go right-to-left.
                </bdo>

Atributos

Atributos Valor Descrição
dir ltr
rtl		 Requerido. Especifica a direção do texto dentro da tag <bdo>
