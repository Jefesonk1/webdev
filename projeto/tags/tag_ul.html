Definição e Uso

A tag <ul> define uma lista não ordenada (com marcadores).
Use a tag <ul> junto com a tag <li> para criar listas não ordenadas.

Exemplo

Uma lista HTML não ordenada:

            
                <ul>
                    <li>Coffee</li>
                    <li>Tea</li>
                    <li>Milk</li>
                </ul>

Atributos

A tag <ul> suporta os Atributos Globais e os Atributos de Eventos.
