Definição e Uso

A tag <select> é usada para criar uma lista drop-down.
A tag <option> dentro da tag <select> define uma opção disponível na lista.
Dica: Sempre adiciona a tag <label> para melhor acessibilidade!

Exemplo

Cria uma lista drop-down com quatro opções:

            
                <label for="cars">Choose a car:</label>

                <select id="cars">
                    <option value="volvo">Volvo</option>
                    <option value="saab">Saab</option>
                    <option value="mercedes">Mercedes</option>
                    <option value="audi">Audi</option>
                </select>

Atributos

Atributos Valor Descrição
autofocus autofocus Especifica que a lista drop-down deve automaticamante receber foco quando a página carregar
disabled disabled Especifica que a lista drop-down deve ser desativada
form form_id Define qual forma a lista drop-down pertence
multiple multiple Especifica que múltiplas seleções são permitidas
name name Define o nome para lista drop-down
required required Especifica que o usuário deve selecionar um valor antes de submeter o formulário
size number Define o número de opções visíveis em uma lista drop-down
