Definição e Uso

A tag <option> define uma opção em uma lista de seleção.
A tag <option> vai dentro de uma tag <select> ou <datalist>.

Exemplo

Uma lista drop-down com quatro opções:

            
                <label for="cars">Choose a car:</label>
                
                <select id="cars">
                    <option value="volvo">Volvo</option>
                    <option value="saab">Saab</option>
                    <option value="opel">Opel</option>
                    <option value="audi">Audi</option>
                </select>

Atributos

Atributos Valor Descrição
disabled disabled Especifica que uma opção deve ser desativada
label text Especifica um rótulo curto para uma opção
selected selected Especifica que a opção deve ser pré-selecionada quando a página carregar
value text Especifica o valor a ser enviado ao servidor
