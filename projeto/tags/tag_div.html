Definição e Uso

A tag <div> define uma divisão ou uma seção em um documento HTML.
A tag <div> é frequentimente usada como um container para outros elementos HTML para estiliza-los com CSS ou fazer certas tarefas com JavaScript.

Exemplo

Uma seção em um documento que irá ter um fundo azul claro:

            
                <div style="background-color:lightblue">
                    <h3>This is a heading</h3>
                    <p>This is a paragraph.</p>
                </div>

Atributos

A tag <div> suporta os Atributos Globais e os Atributos de Eventos.
