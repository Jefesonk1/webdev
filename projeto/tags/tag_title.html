Definição e Uso

A tag <title> é necessária em todos os documentos HTML e define o título do documento.
O elemento <title>:

  • define um título na barra de ferramentas do navegador
  • fornece um título para a página quando ela é adicionada aos favoritos
  • exibe um título para a página nos resultados do mecanismo de busca

Exemplo

Defina um título para o seu documento HTML:

            
                <!DOCTYPE html>
                <html>
                
                    <head>
                        <title>HTML Reference</title>
                    </head>
                    
                    <body>
                        The content of the document......
                    </body>
                    
                </html>

Atributos

A tag <title> suporta os Atributos Globais e os Atributos de Eventos.
