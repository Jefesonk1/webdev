Definição e Uso

A tag <ol> define uma lista ordenada. Uma lista ordenada pode ser numerica ou alfabética.
Use a tag <li> para definir itens da lista.

Exemplo

Duas diferentes listas ordenadas:

            
                <ol>
                    <li>Coffee</li>
                    <li>Tea</li>
                    <li>Milk</li>
                </ol>
              
                <ol start="50">
                    <li>Coffee</li>
                    <li>Tea</li>
                    <li>Milk</li>
                </ol>

Atributos

Atributos Valor Descrição
reversed reversed Especifica que a ordem da lista deve ser descendente (9,8,7...)
start number Especifica o valor do índice inicial da lista
type 1
A
a
I
i		 Especifica o tipo de marcador para usar na lista
