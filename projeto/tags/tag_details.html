Definição e Uso

A tag <details> especifica detalhes adicionais que o usuário pode visualizar ou esconder.
A tag <details> pode ser usada para criar widget interativo que o usuário pode abrir ou fechar.
Qualquer tipo de conteúdo pode ser colocado dentro da tag <details>.
O conteúdo de uma tag <details> não deve ser visível, a menos que o atributo open estiver definido.

Exemplo

Utilizado a tag <details>:

            
                <details>
                    <summary>Copyright 1999-2018.</summary>
                    <p> - by Refsnes Data. All Rights Reserved.</p>
                    <p>All content and graphics on this web site are the property of the company Refsnes Data.</p>
                </details>

Atributos

Atributo Valor Descrição
open open Especifica que os detalhes devem ser visíveis (open) para o usuário
