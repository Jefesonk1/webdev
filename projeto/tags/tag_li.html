Definição e Uso

A tag <li> define um item de lista.
A tag <li> é usada em listas ordenadas (<ol>), não ordenadas(<ul>) e listas de menu (<menu>).

Exemplo

Uma lista ordenada (<ol>) e uma lista não ordenada (<ul>):

            
                <ol>
                    <li>Coffee</li>
                    <li>Tea</li>
                    <li>Milk</li>
                </ol>

                <ul>
                    <li>Coffee</li>
                    <li>Tea</li>
                    <li>Milk</li>
                </ul>

Atributos

Atributos Valor Descrição
value number Especifica o valor de um item da lista. Os próximos itens da lista incrementarão daquele número (apenas para listas ordenadas <ol>)
