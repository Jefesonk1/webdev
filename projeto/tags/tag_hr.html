Definição e Uso

A tag <hr> define uma parada de um tema em uma página HTML (por exemplo uma mudança para um assunto fora de contexto).
A tag <hr> é usada para separar conteúdo(ou definir uma mudança) em uma página HTML.

Exemplo

Use a tag <hr> para definir uma mudança temática no conteúdo:

            
                <h1>HTML</h1>
                <p>HTML is a language for describing web pages.....</p>

                <hr>

                <h1>CSS</h1>
                <p>CSS defines how to display HTML elements.....</p>

Atributos

A tag <hr> suporta os Atributos Globais e os Atributos de Eventos.
