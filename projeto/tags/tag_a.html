Definição e Uso

A tag <a> define um hyperlink, que é usada para linkar uma página em outra. O seu atributo mais importante é o href, que indica o destino do link. Por padrão, os links aparecerão da seguinte forma em todos os navegadores:

  • Um link nao visitado é sublinhado e azul
  • Um link visitado é sublinhado e púrpura
  • Um link ativo é sublinhado e vermelho

Exemplo

Um link para W3Schools.com:

            
            <a> href="https://www.w3schools.com">Visit W3Schools.com!</a>
